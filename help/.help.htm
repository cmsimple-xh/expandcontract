Expandcontract_XH 0.7

For CMSimple_XH from 1.6.4 or higher

This plugin creates links, which on click expand the content of the linked page. Another click will contract this content again. Without java script the links act a standard links.

New in 0.7

  • Expand-links can be inline buttons instead of standard links. These buttons align next to each other even if one of them is clicked.
  • A comma separated list of links can be gives as 1st argument

The pages linked via Expandcontract are either

  • all hidden subpages under the page where the plugin is called (default setting), e.g.:
    {{{expand}}}
    The plugin either creates a button line or <p>-paragraphs for every created link.

  • the page(s) given as 1st parameter
    {{{expand 'page name'}}}
    {{{expand 'page1,page2,page3,etc...');}}}
  • a single page with special link text, e.g.
    {{{expand('page name','link text');}}}
    In linking a single page the plugin will not create a new paragraph. Thus the plugincall can be entered into normal text.

    Please note: The content of the linked page is hidden in a <div> after the plugin call. In a <p> paragraph divs are not allowed and the browser will generate a new paragraph after the div plus a small html error, because or the remaining </p>. All this is prevented by formatting the paragraph as <div> and not as <p>.

Plugin config

Set in plugin config

  • whether link style or buttons should be used,
  • whether the expanded content should show its heading or not,
  • whether at the bottom of the expanded content a "Close" button should be put or not.
  • whether the height of the expanded area is limited and gets a vertical scroll bar.

In the plugin call config settings can be overwritten (use 1 and 0 without quotation marks or text with quotation marks). There are 6 optional parameters:
{{{expand 'pagename(s)', 'link text', 'headings-on/off', 'Close-button-on/off', 'max-height', 'button-style-on/off'}}}

Examples

{{{expand('Terms and Conditions','click here for our Terms and Conditions', 1, 1, '300px');}}}
The words click here for our Terms and Conditions are shown as link. A click will expand the content of the page Terms and Conditions with heading, "close" button and limited height of 399 px (regardless of config settings in the plugin backend).

{{{expand('Workshop Timetable','click here for more details');}}}
The words click here for more details are shown as link. A click will expand the content of the page Workshop Timetable with heading, "close" button and height limit as set in the plugin backend.

Installation and update

Unzip, copy the plugin in your plugin folder or overwrite existing plugin and go to the backend. This will trigger the creation or update of the config values.
